# Arduino LED Blink

## Project Overview
This project demonstrates the basic LED blinking program using an Arduino Uno. It is one of the fundamental experiments for learning Embedded Systems and microcontroller programming.

## Objective
- Learn Arduino programming basics.
- Understand digital output pins.
- Control an LED using Arduino.
- 
## Components Required
- Arduino Uno
- LED
- 220Ω Resistor
- Breadboard
- Jumper Wires
- USB Cable

## Software Used
Arduino IDE

## Circuit Connection
| Arduino Pin | Connection |
|-------------|------------|
| D13 | LED Anode (+) |
| GND | LED Cathode (-) through 220Ω resistor |

## Working Principle
The Arduino configures pin 13 as an output. The LED is turned ON for one second and then OFF for one second repeatedly, creating a blinking effect.

## Program
The Arduino source code is available in:
LED_Blink.ino

## Output
The LED connected to digital pin 13 blinks continuously with a delay of one second.

## Learning Outcome
- Arduino Programming
- Digital Output
- pinMode()
- digitalWrite()
- delay()
